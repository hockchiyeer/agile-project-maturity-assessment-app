# Release Notes

## Latest Release

The Agile Project Maturity Assessment App is now available at [https://gemini.google.com/share/934a5126ecdb](https://gemini.google.com/share/934a5126ecdb).

With this release, teams can rate `1-5` maturity scores across different agility disciplines over time using periodic assessment dates such as monthly or quarterly review points.

This release focuses on improving data portability, flexibility, traceability, and maturity scoring over time.

## What's New

### 1. Slide Deck Export

Maturity assessment results can now be exported as PowerPoint (`PPTX`) or PDF slide decks for easier reporting and stakeholder sharing.

### 2. Custom Discipline Creation

Users can now define new disciplines beyond the default `A-H` categories, making the assessment model more adaptable to different teams and delivery contexts.

### 3. Inline Discipline Renaming

Discipline names can now be edited directly in the interface. Updates automatically propagate across tables, filters, dropdowns, scorecards, and radar chart labels.

### 4. Dynamic Question Management

A new `Add Question` modal allows users to create additional principles and assign them to existing disciplines or newly created ones.

### 5. Adaptive Radar Chart

The radar chart now updates dynamically based on the active dataset, including its scale and displayed labels.

### 6. JSON-Based Data Handling

The app now supports opening, saving, linking, and downloading assessment data in JSON format to improve portability and reuse.

### 7. Change Tracking and Audit Log

Traceability has been expanded through preserved original and current datasets, plus an audit log that captures additions, edits, deletions, renames, and resets.

### 8. Date-Based Score Tracking

Users can now record `1-5` maturity scores across multiple assessment dates, with up to `12` dated entries maintained in a single timeline.

### 9. Assessment Timeline Management

Assessment snapshots can now be added, updated, removed, and switched directly within the app while keeping the dashboard aligned to the selected date.

### 10. Responsive Mobile and Desktop Layout

The interface has been improved so timeline controls, tables, scorecards, and charts adapt more cleanly across both desktop and mobile screen sizes.

### 11. Date Comparison Graphs

A dedicated comparison section now visualizes score changes between dates using both overall and discipline-level views.

### 12. Switchable Comparison Chart Types

Date comparison visuals can now be toggled between bar graphs and line graphs depending on the reporting need.

### 13. Enhanced Export Coverage

PDF and PowerPoint exports now include date comparison visuals, selected chart types, and dated comparison summaries in addition to the existing maturity views.

## Release Summary

This release turns the app into a more flexible maturity tracking tool by combining dated assessments, richer exports, portable JSON-based data handling, and stronger traceability. Teams can now evaluate progress over time, customize the framework to their context, and generate clearer reporting outputs from the same dashboard.
